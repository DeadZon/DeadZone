## How to Add a Device

1. Confirm SoC (Snapdragon or MTK — do not guess).
2. Confirm marketing name.
3. Create `registry/devices/<soc>/<codename>.yml`.
4. Add entry to `registry/device_groups/<soc>.yml` with `status: ready`.
5. Add `"<codename> | <Marketing Name>"` to the correct workflow device options.
6. Test: `python -m factory.cli plan --device <codename> --soc <soc> ...`
